Что сделано — Service Desk (Корпоративная система тикетов)
=========================================================

Стек: React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui + Express + MySQL + Socket.io

ЗАПУСК

  Требования: Node.js 18+, MySQL (Laragon / Docker)

  Быстрый старт (локально):
    cp server\.env.example server\.env
    cd server && npm run dev
    cd .. && npm run dev
    → http://localhost:5173 (фронтенд)
    → http://localhost:4000/api (API)
    → http://localhost:4000/api/docs (Swagger)

  Docker (prod):
    docker compose up -d --build
    → http://localhost

  E2E тесты:
    npx playwright install chromium
    npm run test:e2e

  DevOps логин: POST /api/auth/dev-login (без пароля, пользователь id=1)
