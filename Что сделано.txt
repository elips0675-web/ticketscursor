ЗАПУСК
━━━━━━━━━━━━━━━━━━━━━━━━━

Требования: Node.js 18+, MySQL (через Laragon)

1. Импорт БД:
   mysql -u root -p < server\seed.sql

2. Настройка .env:
   copy server\.env.example server\.env
   (прописать DB_PASSWORD если нужно)

3. Запустить всё:
   запустить-все.bat

Или вручную:
   npm run dev        → http://localhost:5173
   cd server && npm run dev  → http://localhost:4000

Dev-логин: POST /api/auth/dev-login (без пароля)
