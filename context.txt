## Проблемы (решённые)

### MySQL не запущен → 500 на всех страницах
- **Причина**: Laragon был запущен, но MySQL-сервис не стартовал
- **Решение**: перезапущен Laragon → MySQL поднялся
- **Дополнительно**: `deleted_at`, `last_active`, `event_outbox` не существовали в БД — добавлены вручную

### Prisma client не синхронизирован со схемой
- **Причина**: после мержа 10 коммитов (soft-delete, outbox) `prisma generate` не был запущен
- **Решение**: `npx prisma generate` + ALTER TABLE для добавления колонок

## Admin Operations — `/admin/settings`

Страница "Настройки" теперь содержит блок **"Операции"**:

| Операция | Что делает | API |
|----------|-----------|-----|
| **Redis** | Показывает статус (ping), позволяет ввести/сохранить REDIS_URL | `GET /admin/settings/redis-status`, `PUT /admin/settings/redis` |
| **Backup** | Запускает `scripts/backup-mysql.ps1` (mysqldump, retention 7d) | `POST /admin/settings/backup` |
| **Seed** | Запускает `npm run db:seed` в server/ | `POST /admin/settings/seed` |
| **Geo** | Запускает FULLTEXT миграцию (заглушка) | `POST /admin/settings/geo` |

Файлы: `server/src/routes/admin.js`, `src/pages/AdminSettings.tsx`, `scripts/backup-mysql.ps1`

## Feature Flags — реализация (Этап 29)

**Backend:**
- Миграция `server/migrations/20260720_feature_flags.js` — таблица `feature_flags` (key PK, enabled, description, updated_at)
- `GET /api/admin/features` — кэшируется 30с через `cacheMiddleware`
- `PUT /api/admin/features` — bulk update флагов, инвалидирует кэш
- Prisma модель `feature_flags` в `schema.prisma`

**Frontend:**
- `src/hooks/useFeature.ts` — хук `useFeature(key)` возвращает boolean (default true), использует React Query со staleTime 30s
- `FeatureFlagsSection` в `AdminSettings.tsx` — toggle-переключатели для каждого флага, кнопки Save/Reset
- i18n ключи `admin.features`, `admin.featuresSubtitle`, `common.reset`

**Тесты:**
- Сервер: 5 тестов (GET defaults, PUT update, GET after update, non-admin rejection, invalid body)
- Клиент: 3 теста (useFeature, useAllFeatures) + 2 теста (feature section rendering, toggle)
- Фронт: 52 файла / 372 теста, сервер: 25 файлов / 352 теста
